# alonememo
1. '포스팅 박스 열기' 버튼을 누르면 내용을 입력할 수 있는 폼이 나타납니다.
2. '아티클 URL'에 네이버 영화 페이지의 특정 영화 url을 입력하고 코멘트를 적은 뒤 '기사 저장' 버튼을 누릅니다.
3. 영화 포스터와 제목, 내용을 크롤링 해서 DB에 저장하고 하단에 나타냅니다.
